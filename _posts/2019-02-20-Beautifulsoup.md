##描述
repr() 函数将对象转化为供解释器读取的形式。

##语法
以下是 repr() 方法的语法:

repr(object)
参数
object -- 对象。
返回一个对象的 string 格式。

实例

```
s = 'RUNOOB'
repr(s)
>>> "'RUNOOB'"
dict = {'runoob': 'runoob.com', 'google': 'google.com'};
repr(dict)
>>> "{'google': 'google.com', 'runoob': 'runoob.com'}"
```
