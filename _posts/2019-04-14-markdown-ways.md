先来看一下列表复制法
```
a = [1, 2, 3]
b = a
a.append(4)
print a
>>> [1, 2, 3, 4]
print b
>>> [1, 2, 3, 4]
```
我们并没有复制a所指引的列表。我们只是创建了一个新的标签b，然后将其指向a所指向的列表。
